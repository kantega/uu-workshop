Ein skjermleser les inn alle element når den laster sida, så den får ikkje med seg viss eit element plutseleg blir populert, som ofte skjer med feilmeldinger. Aria-live er ein attributt du kan legge til på element som kan endre seg. aria-live="presentational" vil gjøre at skjermleseren les opp feilmeldingen når den oppstår.
